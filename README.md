# 🔐 Cipher Decryption Toolkit

A comprehensive Python toolkit for decrypting classical ciphers using brute-force and dictionary-based approaches. This project was created with the goal of decrypting unsolved ciphers from Black Ops 3 Zombies, but can be used for any classical cipher decryption tasks.

## 📋 Supported Ciphers

The toolkit includes decryptors for the following cipher types:

### Vigenère Family
- **Standard Vigenère** (`vigenere-decryptor/`) - Classic Vigenère cipher with keyword
- **Keyed Vigenère** (`vigenere-keyed-decryptor/`) - Vigenère with custom alphabet based on keyword
- **Autokey Vigenère** (`vigenere-autokey-decryptor/`) - Vigenère with plaintext as key
- **Keyed Autokey Vigenère** (`vigenere-keyed-autokey-decryptor/`) - Autokey with custom alphabet
- **Progressive Key Vigenère** (`vigenere-progressivekey-decryptor/`) - Vigenère with incrementing key
- **Keyed Progressive Key Vigenère** (`vigenere-keyed-progressivekey-decryptor/`) - Progressive key with custom alphabet

### Other Classical Ciphers
- **Caesar Cipher** (`caesar-decryptor/`) - Simple shift cipher (brute force all 25 shifts)
- **Keyed Caesar** (`caesar-keyed-decryptor/`) - Caesar with custom alphabet
- **Atbash Cipher** (`atbash-decryptor/`) - Simple substitution cipher
- **Beaufort Cipher** (`beaufort-decryptor/`) - Beaufort cipher variant
- **Beaufort Variant** (`beaufort-variant-decryptor/`) - Modified Beaufort cipher

## 🚀 Quick Start

### Prerequisites
- Python 3.6+
- Required Python packages: `langdetect`

Install dependencies:
```bash
pip install langdetect
```

### Setup

1. **Create required files in the root directory:**

   - `keywords.txt` - Contains keywords for decryption attempts
     - Download from: https://docs.google.com/spreadsheets/d/1R79Y2xCuyJf9sYHrPODwJEcLWe-yxU8McP22n97p5p4/edit?usp=sharing
     - Copy all keywords to `keywords.txt`
   
   - `dictionary.txt` - English word dictionary for scoring
     - Download from: https://raw.githubusercontent.com/dwyl/english-words/master/words_alpha.txt
     - Copy all words to `dictionary.txt`
   
   - `ciphertext.txt` - Your encrypted text to decrypt
     - Place your ciphertext in this file

2. **Run a decryption script:**
   ```bash
   cd vigenere-decryptor
   python vigenere.py
   ```

3. **Sort and rank results:**
   ```bash
   cd ..
   python sort_output.py
   ```
   When prompted, enter the folder name (e.g., `vigenere-decryptor`)

## ⚙️ How It Works

### Decryption Process
1. Each script reads `ciphertext.txt` from the root directory
2. Uses keywords from `keywords.txt` as keys or to create custom alphabets
3. Attempts decryption with all possible key combinations
4. Outputs results to `decrypted_output.txt` in the script's directory

### Ranking and Filtering
The `sort_output.py` script:
- Reads `decrypted_output.txt` from the specified folder
- Scores each line based on English word ratio
- Filters out non-English text using language detection
- Ranks results by legibility score
- Outputs top candidates to `sorted_output.txt`

### Scoring Criteria
- **Word Ratio**: Percentage of words that match the English dictionary
- **Language Detection**: Uses `langdetect` to verify English text
- **Minimum Requirements**: Lines must have at least 4 words and word ratio > 0.3

## 🛠️ Customization

### Adding New Ciphers
1. Create a new directory: `new-cipher-decryptor/`
2. Create `new_cipher.py` following the pattern of existing scripts
3. Ensure it reads from `../ciphertext.txt` and `../keywords.txt`
4. Output to `decrypted_output.txt` in the same directory

### Modifying Scoring
Edit `sort_output.py` to adjust:
- `word_ratio` threshold (currently 0.3)
- Minimum word count (currently 4)
- Language detection settings
- Output format

### File Configuration
- **keywords.txt**: Add/remove keywords for decryption attempts
- **dictionary.txt**: Modify English word list for scoring
- **ciphertext.txt**: Replace with your encrypted text

## 🔧 Troubleshooting

### Common Issues
1. **File not found errors**: Ensure `keywords.txt`, `dictionary.txt`, and `ciphertext.txt` exist in root directory
2. **Import errors**: Install required packages with `pip install langdetect`
3. **Empty results**: Check that ciphertext is properly formatted and keywords are appropriate
4. **Poor ranking**: Adjust scoring thresholds in `sort_output.py`

### Performance Tips
- Use smaller keyword lists for faster processing
- Run multiple cipher types in parallel
- Monitor disk space for large output files
